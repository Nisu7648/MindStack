## 🎯 COMPLETE SYSTEM OVERVIEW

```
USER TRANSACTION
       ↓
JOURNAL ENTRY (Double-Entry)
       ↓
    ┌──────────────────────────────────┐
    │                                  │
    ↓                                  ↓
LEDGER (Account-wise)          SUBSIDIARY BOOKS
    │                          (Books of Original Entry)
    │                                  │
    ├─ Cash A/c                       ├─ 1. Purchase Book
    ├─ Bank A/c                       ├─ 2. Sales Book
    ├─ Sales A/c                      ├─ 3. Purchase Return Book
    ├─ Purchase A/c                   ├─ 4. Sales Return Book
    ├─ Debtors A/c                    ├─ 5. Cash Book
    ├─ Creditors A/c                  ├─ 6. Bank Book
    └─ All Accounts                   ├─ 7. Petty Cash Book
                                      ├─ 8. Bills Receivable Book
                                      └─ 9. Bills Payable Book
```

## 📖 LEDGER SYSTEM

### **Format:**
```
┌──────────────┬────────────────────┬──────────────┬──────────────┬──────────────┬──────────────┐
│     Date     │    Particulars     │  Voucher No. │   Debit (₹)  │  Credit (₹)  │  Balance (₹) │
├──────────────┼────────────────────┼──────────────┼──────────────┼──────────────┼──────────────┤
│              │                    │              │              │              │              │
└──────────────┴────────────────────┴──────────────┴──────────────┴──────────────┴──────────────┘
```

### **Example: Cash Account Ledger**
```
═══════════════════════════════════════════════════════════════════════════════
                         LEDGER ACCOUNT - CASH A/C
                         Account Code: 10001
═══════════════════════════════════════════════════════════════════════════════

┌──────────────┬────────────────────┬──────────────┬──────────────┬──────────────┬──────────────┐
│     Date     │    Particulars     │  Voucher No. │   Debit (₹)  │  Credit (₹)  │  Balance (₹) │
├──────────────┼────────────────────┼──────────────┼──────────────┼──────────────┼──────────────┤
│ 15-Dec-2024  │ To Sales A/c       │ SAL-2425-001 │   11,800.00  │              │  11,800.00 Dr│
│ 17-Dec-2024  │ By Purchase A/c    │ PUR-2425-001 │              │   23,600.00  │  11,800.00 Cr│
│ 19-Dec-2024  │ By Bank A/c        │ CON-2425-001 │              │ 1,00,000.00  │1,11,800.00 Cr│
├──────────────┼────────────────────┼──────────────┼──────────────┼──────────────┼──────────────┤
│              │ Total:             │              │   11,800.00  │ 1,23,600.00  │1,11,800.00 Cr│
└──────────────┴────────────────────┴──────────────┴──────────────┴──────────────┴──────────────┘

Summary:
- Total Entries: 3
- Total Debit: ₹11,800.00
- Total Credit: ₹1,23,600.00
- Closing Balance: ₹1,11,800.00 Cr
```

### **Ledger Balancing Rules:**

**Debit Balance (Dr):**
- Assets (Cash, Bank, Inventory, Fixed Assets)
- Expenses (Rent, Salary, Electricity)
- Losses (Loss on Sale of Assets)

**Credit Balance (Cr):**
- Liabilities (Creditors, Loans, GST Payable)
- Capital (Owner's Capital, Reserves)
- Income (Sales, Interest Income, Commission)
- Gains (Profit on Sale of Assets)

## 🔄 COMPLETE FLOW

### **Example: Credit Sale Transaction**

**1. User enters transaction:**
```javascript
{
  type: 'CREDIT_SALE',
  amount: 59000,
  customerName: 'ABC Pvt Ltd',
  invoiceNumber: 'INV-002',
  gstRate: 18
}
```

**2. System creates:**

**a) Journal Entry:**
```
Dr. ABC Pvt Ltd A/c          ₹59,000
    Cr. Sales A/c                    ₹50,000
    Cr. GST Output CGST A/c          ₹4,500
    Cr. GST Output SGST A/c          ₹4,500
```

**b) Ledger Postings:**
- ABC Pvt Ltd A/c: Dr ₹59,000
- Sales A/c: Cr ₹50,000
- GST Output CGST A/c: Cr ₹4,500
- GST Output SGST A/c: Cr ₹4,500

**c) Sales Book Entry:**
```
Date: 16-Dec-2024
Customer: ABC Pvt Ltd
Invoice: INV-002
Amount: ₹50,000
GST: ₹9,000
Total: ₹59,000
```

**3. User can:**
- View ledger of ABC Pvt Ltd A/c
- View Sales Book for December
- Generate PDF of Sales Book
- Generate PDF of ABC Pvt Ltd Ledger
